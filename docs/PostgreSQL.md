# PostgreSQL

PostgreSQL auch Postgres genannt ist ein objektrelationales Datenbankmanagementsystem.

## SQL

list all databases:
`\l `

select database:
`\c [dbname]`

show tables:
`\dt`

show only public tables:
`\dt public.`

create database:
`CREATE DATABASE testdb;`

drop database:
`DROP DATABASE testdb;`
