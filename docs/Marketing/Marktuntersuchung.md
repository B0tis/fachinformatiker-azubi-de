# Marktuntersuchung

Eine **Marktuntersuchung** hat folgende Aufgaben:

* Marktprognose (Ermittlung von Markttrends)
* Marktforschung
	* Bedarfsforschung
	* Produktanalyse
	* Konkurrenzforschung
	* Marktanalyse
	* Kundenanalyse
	* Markbeobachtung
* Markterkundung (Meistens Betriebsintern anhand von Kennzahlen und anderen Information von verschiedenen Abtl.)
