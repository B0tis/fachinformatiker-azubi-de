# Aktivitätsdiagramm

Aktivitätsdiagramm wird für die Aktivitäten verwendet. Es stellt einen Ablauf einer Anwendung/Aufgabe dar. Dabei ist ähnlich aufgebaut wie ein Programmablaufplan.
